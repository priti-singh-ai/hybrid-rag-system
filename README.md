# Hybrid RAG System

Enterprise-style Retrieval-Augmented Generation (RAG) system using:

- PGVector
- LangChain
- Hybrid Search
- BM25
- FastAPI
- Docker

## Goal

Building a production-style GenAI retrieval system for AI engineering portfolio development.

## Planned Features

- Document ingestion pipeline
- Vector embeddings
- Hybrid retrieval
- Reranking
- API layer
- Docker deployment
